# Responsive Pricing Component

A clean, responsive, and modern pricing table page built using Bootstrap. This component features three distinct payment tiers (Free, Pro, and Enterprise) and is designed to adapt seamlessly across different screen dimensions.



## 🚀 Features

*   **Fully Responsive Layout:** Utilizes Bootstrap's flexible grid system to stack nicely on smaller screens and expand on wider ones.
*   **Clean Card Design:** Clear structural hierarchy separation using headers, content text, and styled call-to-action buttons.
*   **Modern Typography:** Styled elegantly with high readability for pricing metrics and feature lists.

  

## 🛠️ Built With

*   **HTML5:** For page structure.
*   **CSS3:** For custom styling tweaks.
*   **Bootstrap:** For responsive grids, cards, and utility classes.
